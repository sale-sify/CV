Mes Projets

To-do Manager

To-Do Manager est une application web simple de gestion de tâches, développée avec Vue 3 et TypeScript. Elle permet aux utilisateurs de créer, modifier et supprimer des tâches, d’attribuer des statuts, et de stocker toutes les données localement dans le navigateur. Le projet a été réalisé comme prototype pour Code2Work.

Product Man

Product Man est une application conçue pour gérer une liste de produits. Le backend repose sur Node.js, Express et MongoDB Atlas, tandis que le frontend est développé avec React (Vite) et Material UI. L’interface en mode sombre offre une expérience utilisateur élégante et ergonomique, permettant d’ajouter, modifier, supprimer et afficher via un système de formulaires.

Egg Timer

Le projet Minuteur à Œuf est une petite application web ludique réalisée en HTML, CSS et JavaScript, pensée pour allier simplicité et fun. Elle permet de choisir la cuisson idéale, puis lance un compte à rebours animé. Le design coloré et léger donne une touche humoristique à un outil du quotidien.

TicketHack

TicketHack est une application web dynamique développée avec HTML, CSS, JavaScript, Node.js et PostgreSQL. Elle permet de rechercher, filtrer et afficher des trajets de train en temps réel grâce à une interface claire et réactive. Le backend gère les requêtes et la base de données PostgreSQL pour stocker les informations sur les trajets, tandis que le frontend mise sur une expérience fluide et intuitive.

Breakout like

Breakout like est, comme son nom l'indique, un jeu type breakout classique, ou il suffit de deplacer un raquette pour faire rebondir un balle et l'emvoyer casser des briques. Le jeu a ete cree en JS vanilla, utilisant la balise "canvas" du HTML pour y dessiner et faire intereagir les differents elements du jeu (collision / rebondissement etc...)
